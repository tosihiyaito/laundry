# 🧺 ランドリー報告アプリ（試作版 / Phase 1）

サウナ施設の外注ランドリー記録を、**手書きからスマホ入力に置き換える**ためのアプリです。
担当者がスマホで「日付・担当者・洗濯機の大きさ・乾燥機の代金・タオルの枚数（種類別）」を報告し、
一覧と月別の合計を確認できます。タオルは **ジャンボタオル / バスタオル / フェイスタオル** の3種類を種類ごとに記録します。
洗濯機は **大・中・小それぞれの使用回数**を記録でき、1日に複数のサイズ（大と中 など）や同じサイズの複数回もそのまま記録できます。
回数を入れると **洗濯機代が自動計算**されます（大¥1,200・中¥900・小¥600）。さらに「洗濯機代＋乾燥機代」の **合計金額** も自動で表示・集計されます。

---

## 📲 使い方（とてもかんたん）

### 1. アプリを開く
`index.html` を **ダブルクリック** すると、ブラウザ（Chrome・Edge・Safari など）で開きます。
インストールや設定は不要です。

### 2. 3つの画面（下のタブで切り替え）

| タブ | できること |
|------|-----------|
| 📝 **報告** | 日付・担当者・洗濯機の大きさ・乾燥機の代金・タオルの枚数を入力して報告 |
| 📋 **一覧** | 報告した記録を新しい順に表示（✏️編集・🗑️削除もできる） |
| 📊 **集計** | 月ごとの「乾燥機代の合計・タオルの合計枚数・報告件数」を自動計算 |

### 3. スマホで使うには
スマホで使う場合は、後述の **Phase 2（ネット公開）** が必要です。
今の試作版は「まず動きと使い勝手を確かめる」段階のため、開いた端末の中だけで動きます。

> 💡 **試しにスマホで開く方法**：`index.html` を自分宛にメールやLINEで送って、
> スマホで開いても動作を確認できます（データはそのスマホの中だけに保存されます）。

---

## 💾 データの保存について（大切）

- 入力したデータは、**開いているこの端末（ブラウザ）の中だけ**に保存されます。
- 別のスマホやPCには共有されません（管理者がまとめて見るのは Phase 2 から）。
- **CSV書き出し**ボタンで、いつでもExcel／スプレッドシート用のファイルに取り出せます
  （バックアップとしてもおすすめ。Excelで開いても日本語が文字化けしません）。

⚠️ ブラウザの「閲覧履歴・サイトデータの削除」をすると記録も消えます。
大切なデータは定期的に **CSVで書き出して保存** してください。

## ⚙️ 現場に合わせて変える（任意）

`index.html` を編集ソフト（メモ帳でも可）で開き、上の方にある設定部分を書き換えるだけです。

```js
// 担当者の候補（自由入力もできます）
const STAFF = ["前川", "新山"];   // ← 実際のメンバー名に

// 洗濯機の大きさの区分
const WASHER_SIZES = ["大", "中", "小"];   // ← 実際の区分に

// 洗濯機の1回あたりの料金（回数 × この単価で自動計算）
const WASHER_PRICES = { "大": 1200, "中": 900, "小": 600 };   // ← 料金を変えられます

// タオルの種類（種類ごとに枚数を記録。増やしたり減らしたりできます）
const TOWEL_TYPES = ["ジャンボタオル", "バスタオル", "フェイスタオル"];
```

変更に迷ったら、そのまま私（Claude）に「担当者を○○と○○に変えて」と頼んでください。

## 🚀 次のステップ（Phase 2・これから）

試作版を気に入っていただけたら、次は実運用に向けて以下を進めます。

1. **全員のデータを1か所に集約**（Googleスプレッドシート 等）
   → どのスマホから報告しても、管理者が全件まとめて確認できる
2. **ネット公開**（GitHub Pages）
   → 担当者はリンクをブックマークするだけでスマホから使える
3. **管理者用ダッシュボード**
   → 月次レポート・端末をまたいだ共有・クラウドバックアップ
